Overview

What is fullautoresearch?

autoresearch is an open-source framework that gives AI models full autonomy to design, implement, and evaluate machine learning experiments. Each model operates as a complete research agent -- reading data, writing code, debugging failures, and iterating on results -- with no human in the loop.

A

Autonomous Experimentation

Models independently design ML experiments, write training scripts, handle errors, and evaluate results. No human guidance during runs.

4

Four-Model Comparison

Head-to-head evaluation of Claude Sonnet 4, Sonnet 4.6, Opus 4.6, and GPT-4.1 on real ML tasks using identical benchmarks.

R

Reproducible Pipeline

Every experiment is logged, versioned, and reproducible. Setup scripts, pre-flight tests, and structured output for easy analysis.

$

Cost-Aware Design

Detailed cost tracking per experiment and per model. Run hundreds of experiments and know exactly what you spent.

Getting Started

Quick Start Guide

Get running in under five minutes. Each command block can be copied directly.

1
Clone and Setup
# Clone and setup
git clone https://github.com/bmdhodl/fullautoresearch.git
cd fullautoresearch
bash scripts/setup.sh
2
Run Pre-flight Tests
# Run pre-flight tests
bash scripts/test.sh
3
Run with Sonnet 4.6 (Recommended)
# Run with Sonnet 4.6 (recommended)
AUTORESEARCH_DEPTH=8 AUTORESEARCH_BATCH_SIZE=16 \
  bash scripts/run_forever.sh --dataset pubmed --tag my-run
4
Run with Opus 4.6 (More Capable, Higher Cost)
# Run with Opus 4.6 (expensive but smarter)
bash scripts/run_forever.sh --opus --dataset pubmed --tag my-opus-run
5
Run with GPT-4.1 via Azure
# Run with GPT-4.1 via Azure
bash scripts/run_forever.sh --azure gpt-4.1 --dataset pubmed --tag my-azure-run
